================================================================================
                      COMO TESTAR
================================================================================

1️⃣ INSTALAÇÃO & EXECUÇÃO
   $ cd /vercel/share/v0-project
   $ npm install
   $ npm run dev
   → Abrir http://localhost:3000

2️⃣ TESTES MOBILE
   • DevTools → Toggle device toolbar (Ctrl+Shift+M)
   • Testar resoluções: 320px, 480px, 768px, 1024px
   • Rotação: portrait → landscape
   • Toque: hamburger menu, buttons, links

3️⃣ TESTES DE ACESSIBILIDADE
   • Tab: navegação completa
   • Keyboard: Enter para submit
   • Screen reader: VoiceOver (Mac) ou NVDA (Windows)
   • Color contrast: DevTools → Accessibility

4️⃣ TESTES DE FUNCIONALIDADE
   • Home → Clique em "Ver Empregos"
   • Busca → Digite título ou localidade
   • Login → Email com @, senha 6+ chars
   • SignUp → Senha e confirmação devem bater
   • Post Job → Salário deve ser número positivo

5️⃣ VALIDAÇÃO COMPLETA
   Ver: TESTING_GUIDE.md (211 linhas com checklist completo)
